List comprehension¶

evens1 = []          #The long way
for i in range(10):
    if i % 2 == 0:
        evens1.append(i)
print(evens1)
                     #the short way using list comprehensions 
evens2 = [i for i in range(10) if i % 2 == 0]
print(evens2)
>>> [0, 2, 4, 6, 8]

Sets¶

Sets are mutable, and may therefore not be used, for example, as keys in dictionaries.

a = set([1, 2, 3, 4])
b = set([3, 4, 5, 6])
a | b            # Union
>>>{1, 2, 3, 4, 5, 6}
a & b            # Intersection OR a.intersection(b)
>>> {3, 4}
a < b            # Subset or b.issubset(a)
>>> False        #        or b.issuperset(a) to check for superset
a - b            # Difference or a.difference(b)
>>> {1, 2}
a ^ b            # Symmetric Difference or a.symmetric_difference(b)
>>> {1, 2, 5, 6}

Frozensets¶

An immutable form of sets. which can be used as keys for a dictionary

>>> a = frozenset([1, 2, 3])
>>> b = frozenset([2, 3, 4])
>>> a.union(b)
frozenset([1, 2, 3, 4])
# recall that sets cannot contain other sets, but frozen sets can
>>> a = set([1, 2, 3])
>>> b = set([2, 3, 4])
>>> a.add(b)
>>> a.add(frozenset(b))

Methods¶

include: intersection(), union(), symmetric_difference(), difference(), issubset(), isdisjoint() issuperset() and copy()
